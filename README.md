# RM Services - Plateforme de Solutions Professionnelles

![Logo RM Services](public/rm-services-logo.png)

Bienvenue sur le dépôt officiel de **RM Services**, une application web moderne et haute performance conçue pour présenter des services professionnels avec une expérience utilisateur premium.

## Fonctionnalités Clés

- **Entièrement Responsive** : Optimisé pour tous les appareils, du mobile aux écrans ultra-larges.
- **UI/UX Moderne** : Construit avec une esthétique "Modern Dark", incluant du glassmorphisme et des transitions fluides.
- **Performance Optimisée** : Exploite les dernières fonctionnalités de Next.js 16 pour des temps de chargement ultra-rapides.
- **Animations Dynamiques** : Interactions fluides propulsées par `framer-motion`.
- **Orienté Services** : Sections dédiées aux services, projets et documentation professionnelle.

## Stack Technique

- **Framework Principal** : [Next.js 16](https://nextjs.org/) (App Router)
- **Bibliothèque** : [React 19](https://react.dev/)
- **Style** : [Tailwind CSS 4](https://tailwindcss.com/) & [Vanilla CSS](https://developer.mozilla.org/en-US/docs/Web/CSS)
- **Composants** : [Material UI (MUI)](https://mui.com/)
- **Animations** : [Framer Motion](https://www.framer.com/motion/)
- **Icônes** : [Lucide React](https://lucide.dev/) & [MUI Icons](https://mui.com/material-ui/material-icons/)

## Mise en Route

### Prérequis

- Node.js (Version LTS recommandée)
- npm / yarn / pnpm

### Installation

1. **Cloner le dépôt** :
   ```bash
   git clone https://github.com/ngoungouloufabrice/RM-SERVICES.git
   cd rm-services   
   ```

2. **Installer les dépendances** :
   ```bash
   npm install
   ```

3. **Lancer le serveur de développement** :
   ```bash
   npm run dev
   ```

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur pour voir le résultat.

## Structure du Projet

```bash
/app          # Next.js App Router (Pages & Layouts)
/components   # Composants UI réutilisables
/public       # Actifs statiques (images, logos)
/styles       # Styles globaux et configuration Tailwind
```
